# SwiftAPI 隐私策略 (Privacy Policy)

**中文：**
SwiftAPI 是一款本地运行的 API 开发与测试工具。本软件尊重并保护所有用户的个人隐私权。
我们郑重声明：SwiftAPI **不会**在后台收集、存储、传输或分享您的任何个人信息、API 密钥、请求数据或使用习惯。所有的网络请求和数据保存均在您的本地计算机上完成。

**English:**
SwiftAPI is a locally-run API development and testing tool. We respect and protect the personal privacy of all users.
We solemnly declare that SwiftAPI **DOES NOT** collect, store, transmit, or share any of your personal information, API keys, request data, or usage habits in the background. All network requests and data storage are completed entirely on your local computer.
